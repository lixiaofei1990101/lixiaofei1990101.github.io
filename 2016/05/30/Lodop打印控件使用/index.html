Lodop打印控件使用

2016-5-30 周一 15:17

官网地址:http://www.lodop.net/index.html
在这个官方网站包含了很多打印常见的一些案例，只需要在页面查看源码就OK了，下面就LODOP在实际项目中的使用，稍微演示一下。

一、lodop植入到jsp页面

<script type="text/javascript" src="js/print/LodopFuncs.js"></script>
<object id="LODOP_OB" classid="clsid:2105C259-1E0C-4534-8141-A753534CB4CA" width="100%" height="500">
    <param name="Caption" value="内嵌显示区域">
    <param name="Border" value="1">
    <param name="Color" value="#C0C0C0">
    <embed id="LODOP_EM" type="application/x-print-lodop" width="100%" height="500" pluginspage="js/print/install_lodop32.exe"></embed>
</object>

做完上面的这些操作之后，初始化我们的打印控件，比较简单，可以去官网参考一下，这个在官网有。一般情况下，像我们的打印设计和打印的调整都需要建立在这个控件植入的基础之上。

二、使用

var LODOP=getLodop(document.getElementById('LODOP_OB'),document.getElementById('LODOP_EM'));

这里获取打印的操作对象，所有的打印方法都是建立在这个对象之上的，比如一些常用的打印，打印预览，打印内容的元素定位等等。

三、实例

1、批量打印，实现分页打印。

function getTemplate2(printInfo){
    LODOP.SET_PRINT_PAGESIZE(1,1000,1500,"");
    LODOP.ADD_PRINT_LINE(101,0,100,440,0,1);
    LODOP.ADD_PRINT_LINE(191,0,190,440,0,1);
    LODOP.ADD_PRINT_LINE(190,143,0,144,0,1);
    LODOP.ADD_PRINT_TEXT(39,13,127,31,"店铺单号:");
    LODOP.SET_PRINT_STYLEA(0,"FontSize",18);
    LODOP.ADD_PRINT_BARCODE(9,164,212,80,"128A",printInfo.orderNo);
    LODOP.ADD_PRINT_TEXT(114,44,44,59,"A");
    LODOP.SET_PRINT_STYLEA(0,"FontName","微软雅黑");
    LODOP.SET_PRINT_STYLEA(0,"FontSize",35);
    LODOP.SET_PRINT_STYLEA(0,"Alignment",2);
    LODOP.ADD_PRINT_BARCODE(105,155,212,80,"128A",printInfo.billNo);
    LODOP.ADD_PRINT_TEXT(197,15,212,15,"To:"+printInfo.dstcusPerson);
    LODOP.SET_PRINT_STYLEA(0,"FontSize",10);
    LODOP.ADD_PRINT_TEXT(212,16,203,27,printInfo.dstcusAddr);
    LODOP.ADD_PRINT_TEXT(244,18,115,26,printInfo.dstCity);
    LODOP.ADD_PRINT_TEXT(244,170,131,20,printInfo.zipCode);
    LODOP.ADD_PRINT_TEXT(273,18,161,20,"Russian Federation");
    LODOP.SET_PRINT_STYLEA(0,"FontSize",11);
    LODOP.ADD_PRINT_TEXT(291,49,180,20,mobile);
    LODOP.ADD_PRINT_TEXT(292,18,51,15,"Tel：");
    LODOP.SET_PRINT_STYLEA(0,"FontSize",10);
    LODOP.NewPage();
}

如上面所示，printInfo就是我们通过ajax请求获取的打印的对象，如果是集合则可以通过循环的方式调用方法进行分页打印，其中的LODOP.NewPage()就是
用于分页打印的。
2、 批量操作，这里和上面的批量操作稍微有些不同，比如说，我们有很多对象，但是我们要打印在一张A4纸上面，这里lodop有个很好的地方，我们把我们打印的内容按照一定的顺序和版式排列后，然后设置纸张的大小，当打印的部分超过纸张的大小的时候，lodop会自动进行分页的操作。满足这样的需求，具体看下面的代码。

function createPrintTable(jsonData){
    var printtype = $("input[name='printtype']:checked").val();

                "<span style='font-size:25px;'>货铺单号:</span>" +
                "</td>";
        table+="<td style='border:solid 1px black;width:220px;height:50px;font-size:15px;vertical-align:middle;text-align: center;'>" +

                "</td>";
        table+="</tr>";
        table+="<tr>";
        table+="<td style='border:solid 1px black;width:120px;height:50px;font-size:12px;vertical-align:middle;text-align: center;'>" +
                "<span style='font-size:30px;'>A</span>" +
                "</td>";
        table+="<td style='border:solid 1px black;width:220px;height:50px;font-size:15px;vertical-align:middle;text-align: center;'>" +

                printInfo.billNo+

        table+="</tr>";

                printInfo.dstcusAddr+"<br/>"+

                "</td>";
        table+="</tr>";
        table+="</table>";
        table+="</div>";
        obj+=table
    }
    obj+="</div>"
    LODOP.ADD_PRINT_HTM(0,0,2100,2970,obj);
    LODOP.SET_PRINT_STYLEA(0,"FontSize",10);
    if(printtype=="0"){
        LODOP.PRINT();//直接打印
    }else{
        LODOP.PREVIEW();//预览打印
    }
}

上面的jsonData实际也就是后台查询出来的数据，将这些数据遍历出来拼接dom节点的方式，排成A4纸的打印要求。这样就能达到我们需要的效果了。
总之呢，lodop只要我们看到页面的东西，我们都能够打印出来，这就是它的强大之处，灵活之处。里面还有一些很强大的功能，有些事需要付费的，但是如果只是满足我们一般的需求，不付费用的部分应该是足够了。。
